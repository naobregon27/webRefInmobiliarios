# 🏠 Referí y Ganá - Landing Page Profesional

## 📋 Descripción

Landing page moderna, profesional y futurística para la app de referidos inmobiliarios "Referí y Ganá". Diseñada con las mejores prácticas de UX/UI siguiendo el estilo de las apps más exitosas del mundo.

## ✨ Características de la Landing

### 🎨 Diseño Profesional
- **Hero Section** impactante con animaciones y efectos de partículas
- **Gradientes modernos** con colores cyan, blue, purple y gold
- **Glassmorphism** y efectos de blur para un look futurista
- **Animaciones suaves** con Framer Motion en cada sección
- **Responsive** 100% - perfecto en desktop, tablet y móvil

### 📱 Secciones Incluidas

1. **Hero**
   - Título impactante con gradiente animado
   - CTAs prominentes (Descargar App / Ver Cómo Funciona)
   - Mockup de la app con efecto flotante
   - Social proof (120+ referidos, 500+ clientes, $1M+ comisiones)
   - Scroll indicator animado

2. **Features** 
   - 8 características principales con iconos
   - Cards interactivas con hover effects
   - Gradientes únicos por feature
   - Layout en grid responsivo

3. **How It Works**
   - 4 pasos explicados visualmente
   - Layout alternado (zigzag) para mejor lectura
   - Imágenes de la app mostrando cada paso
   - Badges con información adicional

4. **Screenshots**
   - Carousel interactivo de capturas
   - Efecto de blur en imágenes laterales
   - Navegación con flechas y dots
   - Descripciones de cada pantalla

5. **Download CTA**
   - Botones de App Store y Google Play
   - Stats de la app (rating, descargas)
   - QR Code para descarga rápida
   - Trust badges

6. **Footer**
   - Links organizados por categorías
   - Información de contacto
   - Redes sociales con hover effects
   - Copyright y links legales

### 🎭 Efectos y Animaciones

- ✨ Partículas animadas en el background
- 🌊 Efectos de glow y blur
- 📈 Animaciones de entrada (fade in, slide, scale)
- 🎯 Hover effects en todos los elementos interactivos
- 🔄 Scroll-triggered animations
- 💫 Floating effects en los mockups

## 🚀 Stack Tecnológico

- **React 18.2** - Biblioteca de UI
- **Vite 5.0** - Build tool y dev server
- **Tailwind CSS 3.3** - Framework de CSS
- **Framer Motion 10.18** - Animaciones profesionales
- **Lucide React** - Iconos modernos y consistentes

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Si hay problemas, eliminar node_modules y reinstalar
rm -rf node_modules package-lock.json
npm install
```

## 🏃 Ejecución

```bash
# Modo desarrollo
npm run dev

# Build para producción
npm run build

# Preview de producción
npm run preview
```
